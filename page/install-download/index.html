Yui's personal task manager

Download and Install

Project is in alpha stage now, some features might be missing

Yui have 2 clients:

  • Command line interface, free, requires python
  • Graphical user interface (gui).
    • Aplha is free
    • As for GUI requires a lot of effort and testing (even worse - we have to pay 3rd party for certification and distribution) stable releases will be paid, unless we got enought donations to keep it free

Installation

Before you begin

It’s recommended to install git first. Yui use git repository as database behind the scenes. Changes history, revert, push/pull tasks to/from remote server require git to be installed. However, it’s optional and you can do it any time later.

Install Yui-CLI (command line)

Install from python repository (see also pypi.org/project/yui/)

pip install yui

That’s it :)

Install Yui-GUI alpha

  • Download installer for windows 10: Yui-0.0.2-alpha-win10-x64-installer.exe
  • RPM for Fedora/CentOS/RedHat: coming soon
  • DEB for Ubuntu/Debian: coming soon
